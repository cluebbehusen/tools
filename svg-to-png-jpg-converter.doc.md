# svg-to-png-jpg-converter

Convert SVG markup or files to PNG or JPG with sizing, scale, and quality controls.

- Input: paste SVG, upload an SVG file, or load from a URL (CORS required).
- Output: downloadable PNG/JPG with a live preview and open-in-new-tab option.
- Controls: format, width/height, scale, JPG quality, background color, preserve viewBox sizing.
- State: output settings stored in the URL query string for sharing presets.
