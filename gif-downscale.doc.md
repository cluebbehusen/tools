# GIF Downscale Tool

Reduce GIF file sizes by downscaling, skipping frames, and optimizing colors.

## Features

- Resize dimensions (10-100%)
- Skip frames to reduce file size
- Adjust color quality
- Side-by-side preview
- File size comparison

## Usage

1. Upload a GIF file
2. Adjust scale, frame skip, and quality settings
3. Click "Process GIF"
4. Download the optimized result
